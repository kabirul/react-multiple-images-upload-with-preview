# React Multiple Image Upload with Preview example

In this tutorial, I will show you way to build React.js Multiple Image Upload with Preview to a Rest API example. The React App uses Axios and Multipart File for making HTTP requests, Bootstrap for progress bar. You also have a display list of imagesâ€™ information (with download url).


### Set port
.env
```
PORT=8081
```

## Project setup

In the project directory, you can run:

```
npm install
# or
yarn install
```

or

### Compiles and hot-reloads for development

```
npm start
# or
yarn start
```

Open [http://localhost:8081](http://localhost:8081) to view it in the browser.

The page will reload if you make edits.
